# 文件夹功能使用指南

## ✅ 已完成的功能

### 后端
- ✅ 文件夹数据库表和模型
- ✅ 完整的文件夹 API（增删改查）
- ✅ 项目-文件夹关联 API
- ✅ 数据库迁移已执行

### 前端
- ✅ 左侧文件夹侧边栏
- ✅ 文件夹创建/编辑/删除
- ✅ 文件夹颜色选择
- ✅ 项目按文件夹分类显示
- ✅ 创建项目时可选择文件夹
- ✅ 编辑项目时可移动到其他文件夹

## 🎨 界面特点

### 布局
- **左侧侧边栏**：显示所有文件夹和项目数量
- **右侧主区域**：显示当前选中文件夹的项目

### 文件夹功能
1. **创建文件夹**：点击侧边栏顶部的"新建文件夹"按钮
2. **选择颜色**：创建时可选择 6 种颜色（蓝、绿、紫、红、黄、粉）
3. **编辑文件夹**：鼠标悬停在文件夹上，点击编辑图标
4. **删除文件夹**：删除后，文件夹内的项目会移到根目录

### 项目管理
1. **创建项目**：可以选择放入哪个文件夹
2. **编辑项目**：可以移动到其他文件夹或移出文件夹
3. **查看项目**：点击文件夹查看该文件夹内的所有项目
4. **所有项目**：点击"所有项目"查看未分类的项目

## 🚀 启动应用

### 1. 启动后端
```bash
cd backend
python app.py
```

### 2. 启动前端
```bash
cd frontend
npm run electron:dev
```

## 📝 使用流程

1. **登录应用**
2. **创建文件夹**
   - 点击左侧"新建文件夹"
   - 输入名称，选择颜色
   - 点击创建
3. **创建项目**
   - 点击右上角"新建项目"
   - 输入项目信息
   - 选择所属文件夹（可选）
   - 点击创建
4. **管理项目**
   - 点击文件夹查看其中的项目
   - 编辑项目可以移动到其他文件夹
   - 删除文件夹时项目会保留在根目录

## 🐛 故障排除

### 如果文件夹不显示
1. 检查后端是否正常运行
2. 检查浏览器控制台是否有错误
3. 确认数据库迁移已执行

### 如果项目无法移动到文件夹
1. 检查后端 API 是否正常
2. 查看网络请求是否成功
3. 刷新页面重试
